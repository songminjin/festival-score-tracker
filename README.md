# 가나다라마바샤 축제 점수 앱

날짜별 활동 수치와 빙고 미션을 저장하고 누적 현황을 보여주는 정적 웹앱입니다.

## 실행

```bash
cd /Users/minjinmac/ai-test/축제
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173`을 엽니다.

## 저장 규칙

- `점수 저장`, `빙고 저장`: 선택 날짜에 새 기록을 만듭니다.
- `점수 수정`, `빙고 수정`: 선택 날짜의 기존 기록을 현재 입력값으로 교체합니다.
- 다른 날짜의 기록은 총 페이지에 자동 누적됩니다.
- `전체 초기화`: 모든 날짜의 점수와 빙고 기록을 삭제합니다.
- 데이터는 현재 브라우저의 `localStorage`에 저장됩니다.
